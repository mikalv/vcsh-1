vCalc is an advanced scientific calculator based on the Reverse Polish notation used on well-known handheld calculators. Its keyboard driven interface has been thoughtfully designed to be both easy to learn and fast to use. vCalc supports a complete set of scientific operations, binary math in hexadecimal and octal, and has a sophisticated statistics engine that is designed to be robust, even with difficult datasets. vCalc also supports 100 memory registers, has a 10-level "undo" feature, and has the ability to save and load data to disk.

See the tutorial for operational information.

Feature List:

  • Keyboard driven user interface
  • Four basic operations on integer and floating point numbers
  • 100 memory registers with a real time watch capability
  • Exponent, Square Root, %, %T, and Delta %
  • Sine, Cosine, Tangent, and their inverses
  • Log and antilog in bases 10, e, and 2.
  • Floor, Ceiling, Truncate, Round, and Absolute Value
  • Factorial/Gamma Function
  • 1 and 2 variable statistics with multiple concurrent datasets
  • Mean, Standard Deviation, Covariance, Population Covariance, Sum
  • Curve Fitting (Linear, Logarithmic, Exponential, and Power)
  • Support for 64-bit integer math with bitwise AND, OR, XOR, and NOT
  • Display and entry of integers in decimal, hexadecimal, and octal
  • Save and Load to disk
  • Clipboard Cut, Copy, and Paste
  • IEEE-754 compliant math
