Image Extractor FAQ

1) Cannot update an installed image extractor?
A: Shutdown all open Internet Explorer sessions.

2) Cannot extract site?
A: Some sites have programmatic interfaces that the user must click a button to enter. Image Extractor does not perform this user interface action.

3) No thumbnails?
A: Image Extractor does not create thumbnails for the "Fast" option.
