About:
ImageExtractor can help you quick load images that you wish to view. After extracting the images from the site you are visiting (screenshots), ImageExtractor will show you one of several views of the images. These views are also saved to your hard drive so that you can open and view them at a later time.

ImageExtractor also allows you to edit the generated views to get rid of unwanted images. After you finish editing, ImageExtractor will regenerate the views based on your edits.

This utility works in a limited capacity unless you purchase a "Registration Key".

** This utility works with Microsoft Internet Explorer version 5.5 and above. You can download the latest Internet Explorer here.

Getting Started:
1) ImageExtractor has been installed on your system and can be displayed by going to the "View" menu of Internet Explorer. Select "Image Extractor" from the "Explorer Bar" submenu.

 

2) Criteria fields - filters the images and generates the views based on the settings.

Depth - how many links deep to search through the site. A setting of 0 (zero) will extract only the first page you are looking at.
Maximum Images - self explanatory.
Follow Links to Other Sites - ImageExtractor will stay only on the current site if this is not selected.
MinWidth/Height - minimum dimensions of the images to capture.
ThumbSize - dimensions of the Thumbnails generated for the images.
ZoomSize - initial size of the images for the "zoom".
Extract - the extraction button. Starts the extraction process.
Fast - the fast button skips some validation checks and doesn't download the images to your hard drive and create thumbnails. It is roughly twice as fast as the normal extract.

3) Views - different views of the extracted images.

Tiled - shows images scaled normally
Thumbnails - shows thumbnail images
Slideshow - slideshow view
Zoom - a scalable view of the images
Open View - open saved view.
Edit View - allows you to remove and move images from the views. After you are finished with your edits press "save & regenerate" to regenerated views.
Cancel - cancels edit mode.
Save - saves edited view. 

4) Other Settings / Registration
Settings & Registration - allows you to change your storage location and set your registration key (see dialog image below).
Check for Updates - visits the ImageExtractor site.
Email to a Friend - sends a friend a link to ImageExtractor.
Help - pulls up this help file
Tutorials - displays tutorial files
Contact - sends mail to staff
