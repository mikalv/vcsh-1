Extracting Images from a Website

1) Using the web browser select a site that you wish to capture images from.

2) (Optional see 'Setting and Selecting Image Criteria) Set the criteria that you wish to use to filter the images displayed.

3) Press the "extract" button.

The ImageExtractor will proceed to process the website and display progress information. At this time you can continue to browse the web without interfering with the extraction of the site.

Setting and Selecting Image Criteria

At times it is desirable to select the number of images and the dimension of the images that you wish to collect. The depth parameter might as well be set to 1000 and just use 'Maximum Images' to limit the search.

To find 100 images that are bigger than 200x100 (W x H).

1) Set depth to 1000

2) Set 'Maximum Images' to 100.

3) Set 'MinWidth' to 200 and 'MinHeight' to 100

4) Press the "extract" button.

Stopping Image Extraction

After starting an extraction you will be presented with a 'Working' page that displays the current progress of the process.

To stop the collection and acquire the found images.

1) Press "stop".

The collection will stop however there will be a delay of an Indeterminate amount of time while the ImageExtractor downloads and displays the current collection of images.

To abort the collection and discard the found images.

2) Press "cancel".

The collection will stop immediately and you will be returned to the ImageExtractor main panel.

Editing and Saving Views

At times you can capture undesirable images and would want to remove them from the view.

To edit your captured view.

1) Press "edit". You are now in edit mode.

2) Click on undersired images and press the delete button to remove the selected image. Continue removing all undesired images.

3) Press "save" this will regenerate the views and turn off the edit mode.
